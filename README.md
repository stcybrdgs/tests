## Start a GitHub Project
```
$ cd Users/stcyb
$ cd MyProjects
$ mkdir gh-test
$ cd gh-test
$ touch README.md
$ touch notes.txt

# vim READ*
# sudo vim README.md
$ vim README.md
$ vim notes.txt
$ touch .gitignore

$ git init
$ git status
$ git commit -m "first commit"


```

## VIM
Use `SHFT + i` to insert in VIM
Use `esc` to exit editing and allow close options
- `q` to quit
- `w` to write
- `wq` to write and quit
- `q!` to quit without writing
